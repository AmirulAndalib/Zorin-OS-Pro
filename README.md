# Zorin-OS-Pro
This script turns your Zorin OS CORE into Zorin OS PRO

You will have access to all Zorin OS Pro wallpapers and layouts
```bash
#clone this repository
git clone https://github.com/PEAKYCOMMAND/Zorin-OS-Pro

#enter the folder
cd zorin-os-premium

#run the script
sudo ./zorin.sh



```
